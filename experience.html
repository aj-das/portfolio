PROJECTS

SongSync

SongSync aims to solve the challenge of finding the ideal playlist for any mood, offering a unique and tailored musical journey every time. It is a web application designed to revolutionize playlist creation through advanced audio analysis and mood-based curation.

The project involves implementing various methods to sort and manage audio tracks effectively, focusing on features such as energy, tempo, and danceability to ensure each playlist is harmonically and rhythmically cohesive. By integrating with the Spotify API, SongSync accesses users' top artists and creates personalized playlists directly on their profiles. The application employs sophisticated algorithms to match users' selected moods with tracks from their favorite artists, computing mood scores to curate the perfect listening experience.

Design Process: I started development with SongSync by identifying the need for a more personalized music experience that adapts to user mood and preferences. After researching user behavior on music platforms, I initiated a series of brainstorming sessions to outline key features. Early prototypes were built using React and tested for usability. After refining the UI based on feedback, I integrated Spotify API to fetch and analyze user data, ensuring seamless playlist creation. The project's frontend was styled with CSS for an engaging and intuitive user interface.

I invite you to read through the code on GitHub. Check out my LinkedIn post!

Data Structures/APIs/Software Used: Spotify API, FastAPI, SQL, Sorting Algorithms, React/Next.Js, Authentication Systems

MScribe

Along with a team of passionate innovators at the intersection of technology and education, we created M-Scribe, a Chrome extension designed to revolutionize the way students interact with their academic content.

This tool enhances UMich lecture experiences by combining a React frontend with a Flask backend, incorporating cutting-edge AI technologies for real-time lecture analysis and summarization. M-Scribe not only reduces student study time by 40% but also improves comprehension through features like advanced AI-driven note-taking and content summarization. It's about making every study session as productive as possible, freeing up time for what truly matters to students.

Design Process: MScribe was conceived to enhance the educational experience by facilitating better engagement with lecture content. The project began with a thorough analysis of student needs through surveys and one-on-one interviews, which highlighted the demand for advanced note-taking and content summarization tools. We developed prototypes using Plasmo and CSS and refined our approach through multiple rounds of user feedback and functional enhancements.

Check out the GitHub repo to view the code as well as a demonstration of the RAG implementation.

Data Structures/APIs/Software Used: React, Flask, Pinecone, LangChain, Advanced AI (Pinecone + LangChain), PHPSESSID for secure data flow, and custom Chrome extension APIs.
