PROJECTS

SongSync

SongSync aims to solve the challenge of finding the ideal playlist for any mood, offering a unique and tailored musical journey every time. It is a web application designed to revolutionize playlist creation through advanced audio analysis and mood-based curation. The project involves implementing various methods to sort and manage audio tracks effectively, focusing on features such as energy, tempo, and danceability to ensure each playlist is harmonically and rhythmically cohesive. By integrating with the Spotify API, SongSync accesses users' top artists and creates personalized playlists directly on their profiles. The application employs sophisticated algorithms to match users' selected moods with tracks from their favorite artists, computing mood scores to curate the perfect listening experience.

I invite you to read through the code on GitHub. Check out my LinkedIn post!

Data Structures/APIs/Software Used: Spotify API, FastAPI, SQL, Sorting Algorithms, React/Next.Js, Authentication Systems
